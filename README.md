## 这是一份由自己设计并开发的简历

* http://resume.congm.in

Fork或借鉴请注明出处 [@ Cong Min](https://congm.in) . Thx

前提: 已安装`nodejs` `cnpm`
- 0、`cnpm install`安装依赖
- 1、编辑`/src/index.html`文件
- 2、命令行运行`npm run rebuild`
- 3、打开`/index.html`进行预览

> chrome浏览器 - "打印" - 设置无边距 - 即可导出
